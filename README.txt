Simulation java is the network simulation program. It is hardcoded to run
4 computations with 4 input txt files. The names have to be data0, data1,
data2 and data3. All must be txt and be in the root folder of the java
eclipse project. If oyu are using a different IDE your root location may vary.

The report answers the channel size questions, proves speedup and reusability.

The 2 network diagram jpg show the samee network, one of them shows a full manual
run of the netwrok for a small input. The other shows a clean version, use whichever
you prefer.
